# 番茄时钟小程序需求说明书

## 1. 项目概述

### 1.1 项目背景
本项目旨在开发一款基于uniapp和uview2.0的番茄时钟小程序应用，帮助用户通过番茄工作法管理日常任务，提高工作和学习效率。

### 1.2 项目目标
- 提供简洁美观的番茄时钟倒计时功能
- 支持任务管理，包括工作、学习、生活三大类别
- 通过专注计时帮助用户提升工作效率
- 提供沉浸式的用户体验

### 1.3 技术栈
- **框架**: uniapp
- **UI组件库**: uview2.0
- **开发语言**: Vue2
- **数据存储**: 小程序本地存储（uni.setStorage/uni.getStorage）
- **图片资源**: Unsplash API
- **图标**: uView内置图标库

## 2. 功能需求

### 2.1 番茄时钟倒计时功能

#### 2.1.1 倒计时模式
- **工作模式**: 25分钟倒计时
- **短休息模式**: 5分钟倒计时
- **循环功能**: 工作模式结束后自动切换到短休息模式，短休息结束后自动切换回工作模式，形成循环

#### 2.1.2 控制功能
- **开始**: 启动倒计时
- **暂停**: 暂停当前倒计时
- **重置**: 重置倒计时到初始时间

#### 2.1.3 完成提醒
- 倒计时结束时触发提醒功能
- 提醒方式：系统通知/提示音（根据小程序平台能力）

#### 2.1.4 界面要求
- 倒计时时间显示需要专门美化，具有科技感
- 时间数字需要突出显示，易于阅读
- 控制按钮布局合理，操作便捷

### 2.2 任务管理功能

#### 2.2.1 任务列表
- **入口**: 主界面右上角列表图标
- **展示内容**:
  - 任务标题
  - 任务优先级（高/中/低）
  - 任务类型（工作/学习/生活）
- **交互**: 点击任务项进入任务详情页面

#### 2.2.2 任务详情
- **展示内容**（从上到下顺序）:
  1. 任务标题
  2. 任务描述
  3. 任务优先级
  4. 任务分类
  5. 番茄数量（已完成番茄时钟数量）

#### 2.2.3 任务操作
- 查看任务详情
- 编辑任务（后续版本）
- 删除任务（后续版本）
- 添加任务（后续版本）

### 2.3 界面布局要求

#### 2.3.1 整体布局
- **无底部Tab栏**: 应用不包含底部导航栏
- **无顶部导航栏**: 全屏沉浸式展示
- **主界面**: 番茄时钟倒计时界面
- **任务列表**: 独立页面，通过右上角图标进入

#### 2.3.2 设计风格
- **极简主义**: 界面简洁，去除不必要的元素
- **主色调**: 番茄红色（#FF6347或相近色值）
- **配色方案**: 清新柔和的渐变配色，与主色系浑然一体
- **留白设计**: 恰到好处的留白，提升视觉舒适度
- **圆角设计**: 精心打磨的圆角，统一的设计语言
- **微交互**: 细腻的交互动画，提升用户体验
- **视觉比例**: 舒适的视觉比例，符合黄金分割

## 3. 非功能需求

### 3.1 性能需求
- 倒计时精度：秒级精度
- 页面切换流畅，无明显卡顿
- 数据加载快速响应

### 3.2 兼容性需求
- 支持微信小程序平台
- 支持其他uniapp支持的小程序平台（如支付宝小程序、百度小程序等）

### 3.3 数据存储需求
- 使用小程序本地存储保存任务数据
- 数据持久化，应用关闭后数据不丢失
- 支持数据的增删改查操作

### 3.4 用户体验需求
- 界面操作直观，符合用户习惯
- 视觉层次清晰，核心功能突出
- 沉浸式体验，减少干扰元素

## 4. 约束条件

### 4.1 技术约束
- 必须使用Vue2语法，禁止使用Vue3特性
- 必须使用uView2.0组件库
- 图标必须使用uView内置图标，不得使用外部图标库
- 图片资源通过Unsplash API获取

### 4.2 平台约束
- 基于uniapp框架，需遵循uniapp开发规范
- 遵循小程序平台开发规范

### 4.3 设计约束
- 无底部Tab栏和导航栏
- 全屏展示，沉浸式体验
